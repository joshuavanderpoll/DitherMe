# Security Policy

## Supported versions

Only the latest release receives security fixes. Please update before reporting an issue.

## Reporting a vulnerability

Do not open a public issue for security problems.

Report privately in one of two ways:

- Open a [private security advisory](https://github.com/joshuavanderpoll/DitherMe/security/advisories/new) on GitHub, or
- Email opensource@jvdpoll.nl with the details.

Please include:

- What the issue is and the impact.
- Steps to reproduce, or a proof of concept.
- The version and platform you tested on.

## What to expect

- An acknowledgement within a few days.
- An assessment and, if valid, a fix in a following release.
- Credit in the release notes if you want it.

DitherMe is a local desktop app that processes files you open. It does not run a server or send your images anywhere, so most risk is limited to handling untrusted image or template files.
